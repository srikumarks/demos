Instructions
This is a prototype interface to explore raga relationships via bhedams.
  1. You can type a raga name in the "Raga" box to get a list of matches.
  2. .. or you can enter scalar information (arohana only) in the "Scale" box using the "SrRgGmMPdDnN" letters. For example, Shankarabharanm is "SRGmPDNS" and Kharaharapriya is "SRgmPDnS".
  3. The table will display all possible bhedams.
  4. Clicking on a raga name shows known info about it.
Note: Based on discussions with Mr. Ramesh. Data about ragas sourced from Wikipedia's Janya Ragas page.
